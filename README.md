# Usage

## 1. Clone the Repository

```bash
git clone https://github.com/sametcn99/dotfiles.git
cd dotfiles
```

## 2. Run the Setup

Simply run the bootstrap script. It will handle the rest.

```bash
chmod +x setup.sh
./setup.sh
```

## 3. Interactive Process

The script will:

1. Initialize the environment.
2. Check for missing packages against your lists.
3. Display a summary report.
4. Ask for confirmation before proceeding with the installation.
